# Command Format
```
    !            C           #         #           #           #           #

Start CMD    CMD type     Byte 0     Byte 1      Byte 2      Byte 3      Byte 4      ……
 (char)        (char)     (byte)     (byte)      (byte)      (byte)      (byte)
```

# Data Format
```
    !            A            0           #           #           #           #

Start Data    Data type    Sensor ID     Byte 0      Byte 1      Byte 2      Byte 3      ……
 (char)        (char)       (int)       (byte)      (byte)      (byte)      (byte)
```
